# 🎴 Pokemon Deck Builder Portfolio Project

## 📌 Project Overview

The Pokemon Deck Builder is a frontend web application that displays Pokémon data using a public API.

This project is built around the **Pokémon franchise** data and demonstrates modern frontend development skills.

## 🚀 Features

✅ Fetches Pokémon data dynamically from API
✅ Search Pokémon by name or ID
✅ Filter Pokémon by type
✅ Sort Pokémon list
✅ Pagination system
✅ Skeleton loading animation
✅ Card flip interaction
✅ Lazy image loading optimization
✅ Local storage compatibility preparation

## 🛠 Technologies Used

* HTML5
* CSS3 (Modern Dark Theme UI)
* Vanilla JavaScript
* REST API Integration
